# 🌙 AI 새벽배송 🚚

PM들의 AI 활용 사례를 새벽배송 컨셉으로 공유하는 감성적인 웹사이트

## 🎨 컨셉

**컬리 새벽배송**을 모티브로 한 인터랙티브 사례 공유 플랫폼

- **배경**: 밤하늘에서 새벽으로 변하는 그라데이션
- **발표 자료** = 배송지 (집 🏠)
- **댓글** = 배송 후기 (택배상자 📦)
- **방문 추적**: localStorage로 배송 완료 여부 저장

## ✨ 주요 기능

### 1. 배송 지도 (메인 화면)
- **미배송 집** 🏠: 흐릿한 회색, 호버 시 빛남
- **배송 완료** 🏡: 노란 빛나는 집, "배송 완료 ✓" 배지
- **택배상자**: 집 앞에 댓글 수만큼 쌓임

### 2. 배송지 상세 모달
- 배송물 정보 (발표 내용)
- 발송인 정보
- 배송 명세서 (발표 자료 링크)
- 배송 후기 (댓글) 작성

### 3. 새 배송지 등록
- 우측 상단 "📦 새 배송지 등록" 버튼
- 폼 작성 후 구글 시트에 자동 저장

### 4. 감성적인 애니메이션
- ⭐ 별이 반짝이는 밤하늘
- ☁️ 천천히 움직이는 구름
- 📦 택배상자 떨어지는 효과
- 🏡 집이 빛나는 효과
- 트럭 로딩 애니메이션

## 🚀 시작하기

### 1. 파일 구조
```
ai-delivery-theme/
├── index.html      # 메인 HTML (새벽 배경, 별, 구름)
├── app.js          # JavaScript (이미 키값 포함)
├── vercel.json     # Vercel 배포 설정
└── README.md       # 현재 파일
```

### 2. 로컬 실행
```bash
# Python 서버
python3 -m http.server 8080

# 또는 Node.js
npx http-server -p 8080
```

브라우저에서 `http://localhost:8080` 접속

### 3. Vercel 배포
```bash
vercel --prod
```

## 🎯 사용 방법

### 배송지 방문
1. 집 아이콘 클릭
2. 자동으로 "배송 완료" 처리
3. 상세 정보 확인

### 후기 남기기
1. 배송지 모달에서 후기 작성
2. 이름 입력
3. "📦 후기 남기기" 클릭
4. 집 앞에 택배상자 추가!

### 새 배송지 등록
1. "📦 새 배송지 등록" 클릭
2. 정보 입력
3. "🚚 배송 시작!" 클릭

## 🐛 문제 해결

### 배송지가 안 보여요
- 구글 시트가 공개되어 있는지 확인
- 브라우저 콘솔(F12) 에러 확인

### 후기가 저장 안 돼요
- Apps Script가 배포되었는지 확인
- APPS_SCRIPT_URL이 올바른지 확인

### 방문 기록이 사라졌어요
- localStorage 사용 (브라우저 데이터 삭제 시 초기화)
- 정상 동작입니다
